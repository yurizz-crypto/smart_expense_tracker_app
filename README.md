## 💰 Smart Expense Tracker
Smart Expense Tracker is a Flutter-based mobile application designed to help users manage their daily finances with ease. It features a clean, Material 3 interface and real-time data synchronization using Firebase.

## ✨ Features

### Expense Management: 
Create, view, and update transactions with titles, amounts, and dates.

### Categorization: 
Organize spending into predefined groups like Food, Transport, and Shopping using distinct icons and colors.

### Real-time Tracking: 
Monitor expenses as they happen with live updates from a Firestore backend.

### Filtering: 
Quickly view totals and individual entries by specific categories.

### Visual Summaries: 
A dynamic header that calculates and displays your total spending at a glance.

## 🛠️ Tech Stack
    Framework: Flutter
    Backend: Firebase (Firestore)
    State Management: Provider
    Typography: Google Fonts (Montserrat)
    Formatting: Intl package for currency and date styling
